Tutorial para configurar GitHub Pages

Crear un repositorio en GitHub

  1. Ve a GitHub y accede a tu cuenta.
  2. Creas un nuevo repositorio, pónle un nombre.
  3. Añade una descripción y elige que tu repositorio sea público.
  4. Haz clic en "Create Repository".

Subir los archivos

  • El nombre del archivo que deseas subir se debe de llamar "index.html".
  • Selecciona el archivo a GitHub desde la web.

Activar GitHub Pages

  1. Ve a la pestaña "Settings" del repositorio.
  2. En el menú lateral, selecciona "Pages".
  3. En "Source", elige la rama que quieres usar.
  4. Haz click en "Save".
  5. Recarga la página unos segundos y GitHub generará la url.
